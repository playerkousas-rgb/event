# 🎪 童軍活動管理系統 - 系統功能與使用手冊 (Scout Event Management System)

> **COPY RIGHT Scout System**

## 🌟 系統特色總覽

本系統專為香港童軍總會各大型活動（首個試點為 **2026 ISD 港島童軍繽紛日**）打造。它將傳統繁瑣、全公開且版本混亂的靜態 Google Sites 執行手冊，升級為一個具備 **多活動接入、活動密碼保護、7級角色權限、雙向審批與實時試算表聯動** 的現代化網上指揮中心。

---

## 📂 多活動架構說明 (Multi-Event Architecture)
如同 `vsbadge` 支援多個旅團，本系統支援**多個活動獨立接入**（例如：`isd_2026` 港島童軍繽紛日、`isd_2027` 國際大露營等）：
1. **首頁多活動卡片**：使用者進入首頁即可看到所有進行中或即將舉行的活動清單。
2. **私隱活動密碼**：每個活動均可設定專屬密碼，首次接入後會安全儲存於瀏覽器 `localStorage` 中。
3. **靈活擴展**：未來若舉辦 2027 ISD，只需在 Google Sheet 或 `data/events.json` 中增加一筆紀錄即可無縫切換，無需修改前端代碼。

## 👥 7 級權限與角色分級 (7-Level RBAC)
系統嚴格劃分 7 個層級，確保上級能有效管理下級，同時保障內部私隱：
- **Level 1**: `super_admin`（超級管理員 / 我 - 隱藏）
- **Level 2**: `advisor`（顧問，如黃偉安、何家騏）、`chairperson`（主席，如朱家聰）、`admin`（管理員）
- **Level 3**: `vice_chairperson`（副主席，如袁可秀、張佳良、周恒晉、何嘉駿 - 具備物資與膳食審批權）
- **Level 4**: `general_director`（總主任）
- **Level 5**: `director`（主任 - 會議通知默認對象）
- **Level 6**: `staff`（工作人員）
- **Level 7**: `public`（公開訪客）

## 🗂️ 8 大管理卡片模組（含子分頁與審批流）

1. **📋 會議卡片 (Meetings)**
   - 記錄第0次預備會議至第4次籌備委員會議。
   - 頂部置頂「下次會議預告」Banner，管理員可一鍵自動發送會議提醒 Email 給主任或以上級別委員。
2. **👥 工作人員卡片 (Staff) —— 三大子分頁**
   - 📊 **組織架構圖 (Org Chart)**：展示顧問、主席、副主席及各組關係。
   - 👥 **工作人員名單與聯絡 (Directory & Contacts)**：各組主任聯絡電話與職務。
   - 📋 **職務大綱 (Job Duties)**：明確劃分各功能組別的工作範圍。
3. **📁 文件檔案 (Documents)**
   - 儲存總通告、特別通告（Scout for SDGs）、保險清單與財務指引。
4. **💰 財務管理 (Finance)**
   - 內建 2026 ISD 財務預算表與結算總表（附件5）。
   - 清楚標示報價門檻（不超過 $500 免報價、 $500 至 $5,000 口頭報價、 $5,001 以上報價比較）。
5. **活動與攤位 (Activities)**
   - 警察學院步操檢閱、攤位博覽與積極公民工作坊。
6. **膳食管理 (Meals)**
   - 各組提交代訂便當申請，由行政/協調副主席或以上在**批核中心**一鍵批核。
7. **日程表 (Schedule)**
   - 詳列 10 月 4 日從早晨場地劃位、報到、步操檢閱到頒獎典禮的每一個時段。
8. **物資管理 (Supplies)**
   - 總物資清單、車輛通行證申請與協調組審批工作流。

## 🚀 給首次使用者的快速指引 (User Onboarding)

1. **訪客 / 公眾模式**：
   - 打開系統首頁，可直接瀏覽活動日期、集合地點、天氣應變及公開日程與通告。
2. **籌委 / 內部登入**：
   - 點擊右上角 **「電郵登入 / 切換」**，使用您的電郵及自設密碼登入（或使用超管 `sheep` / `0728`）。
   - 成功登入後，點擊活動卡片輸入活動密碼（預設 `1234`）即可解鎖內部 8 大卡片與批核中心。
3. **提交與批核申請**：
   - 各組委員可在「膳食」或「物資」模組點擊「新增紀錄 / 提交申請」。
   - 上級（副主席、顧問、主席）進入 **⭐ 批核中心 (Approval Hub)** 即可實時檢視並點擊「一鍵批核」。
